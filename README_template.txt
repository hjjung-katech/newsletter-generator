Newsletter Generator

===============================================

AI 기반 뉴스레터 생성 도구입니다.

===============================================

빠른 실행

1. 프로그램 실행:
   newsletter_web.exe

2. 웹 접속:
   http://localhost:8000

3. API 키 설정:
   - 웹 페이지 상단 "설정" 탭에서 진행
   - 필수: Gemini API + Serper API

===============================================

문서 안내

함께 제공되는 상세 가이드:

- 빠른시작가이드.txt: 5분 안에 첫 뉴스레터 생성
  처음 사용하시는 분께 권장합니다.

- 사용자가이드.txt: 전체 기능 완전 설명서
  모든 기능을 자세히 알아보려면 이 문서를 참조하세요.

===============================================

필수 API 키

Gemini API (AI 뉴스레터 생성):
https://aistudio.google.com/

Serper API (뉴스 검색):
https://serper.dev/

===============================================

폴더 구조

프로그램 포함 폴더:
- templates/: 뉴스레터 템플릿
- config/: 설정 파일들

자동 생성 폴더:
- output/: 생성된 뉴스레터 저장
- logs/: 프로그램 실행 로그

===============================================

문제 해결

문제 발생 시:
1. logs/ 폴더의 최신 로그 파일 확인
2. 빠른시작가이드.txt의 문제해결 섹션 참조
3. .env 파일의 API 키 확인

===============================================

더 자세한 내용은 빠른시작가이드.txt 또는 사용자가이드.txt를 참조하세요.
